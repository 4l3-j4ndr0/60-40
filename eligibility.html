SALES, TRANSFERS & TITLE RULES

Priority Order for Sale

If an original participant sells:

  1. Qualified Veteran
  2. Qualified First Responder
  3. Open market only if the buyer assumes both Note A and Note B and meets program requirements

Mandatory Assumption

Any buyer/heir/transferee seeking program benefits must:

  • Qualify under eligibility rules
  • Assume both Note A and Note B
  • Comply with primary residence and integrity rules

If both notes are not assumed, then:

  • Title cannot be transferred
  • Possession/keys cannot be delivered
  • Program benefits do not apply

Title Event Settlement Rule

At any title-related event (sale/refinance/transfer outside program rules), the participant must settle what is owed on both Note A and Note B.

FRAUD PREVENTION & PROGRAM INTEGRITY

The primary objective of this initiative is to formally and institutionally present a request to the appropriate authorities for the evaluation and potential implementation of a pilot program focused on stability and dignified access to housing solutions for veterans.

The project is initially developed at the state level, with the intention of serving as a replicable model in other jurisdictions, always within the applicable legal framework and with strict adherence to the privacy of participants.

The program includes safeguards to prevent abuse and speculation, including:

  • Primary residence verification
  • Anti-flipping enforcement
  • Mandatory assumption of both notes
  • Compliance review of caregiver transfers
  • Documented eligibility confirmation

The program is designed to protect participants and the State‚Äînot to enable short-term profit-taking.
